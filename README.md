## About project
This is simple CLI application, where you can play five different games.
## Prerequisites
- Linux, Macos, WSL
- PHP >= 8.0
- Composer >= 2.8.0
- Make
- Git
## Setup
```
$ git clone git@github.com:dlgv24/php-project-45.git

$ cd php-project-45 

$ make install  
```
## How to run games
```
$ make brain-even

$ make brain-calc

$ make brain-gcd

$ make brain-prime
```
